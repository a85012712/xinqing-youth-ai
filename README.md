# 🌈 心晴少年AI

**青少年心理健康智能助手平台**

一个专注于青少年心理健康的AI智能助手，提供心理疏导、情绪分析、知识科普和专业指导。

## 💝 项目愿景

> "每一个青少年都值得被理解，每一种情绪都值得被关注"

心晴少年AI致力于：
- 为青少年提供安全、私密的心理支持
- 帮助识别和理解情绪问题
- 提供科学的心理健康知识
- 引导寻求专业帮助

## ✨ 核心功能

### 💬 智能心理疏导
- 温暖、理解的对话风格
- 流式输出，实时响应
- Thinking状态，展示AI思考过程
- 多轮对话，持续陪伴

### 🎨 情绪识别与分析
- 文字情绪识别
- 情绪状态追踪
- 个性化建议
- 危机预警提示

### 📚 心理健康知识库
- 青少年心理发展知识
- 常见心理问题科普
- 情绪管理技巧
- 压力应对方法

### 🧠 长期记忆系统
- 记住用户的情绪变化
- 追踪对话历史
- 个性化回应
- 持续关注

### 🩺 专业资源引导
- 心理咨询资源
- 危机干预热线
- 专业机构推荐
- 就医指导

## 🚀 快速开始

### 1. 克隆项目
```bash
git clone https://github.com/your-username/xinqing-youth-ai.git
cd xinqing-youth-ai
```

### 2. 安装依赖
```bash
npm install
```

### 3. 配置API
```bash
cp .env.example .env
# 编辑 .env 文件，填入你的API配置
```

### 4. 启动服务
```bash
npm start
```

### 5. 访问应用
打开浏览器访问: http://localhost:3000

## 💬 对话示例

### 情绪支持
```
用户: 我最近总是感到很焦虑，睡不着觉

心晴少年: 我听到你说的了，感到焦虑和睡不着觉确实很辛苦。
你能愿意说出来，这本身就是很勇敢的一步。

焦虑的时候，身体会有很多反应，比如：
- 心跳加速
- 呼吸急促
- 肌肉紧张
- 难以放松

这些都是正常的应激反应，不是你的错。

你愿意告诉我，最近有什么事情让你感到特别焦虑吗？
我们可以一起想想办法。
```

### 知识科普
```
用户: 什么是抑郁症？

心晴少年: 抑郁症是一种常见的心理健康问题，不是"想太多"或"矫情"。

主要表现可能包括：
😔 持续的悲伤或空虚感
😴 睡眠问题（失眠或嗜睡）
🍽️ 食欲变化
😞 对以前喜欢的事情失去兴趣
💭 难以集中注意力
😔 感到自己没有价值

重要的是：
1. 抑郁症是可以治疗的
2. 寻求帮助是勇敢的表现
3. 你不是一个人在面对

如果你或身边的人有这些表现，建议：
- 和信任的人聊聊
- 寻求专业帮助
- 拨打心理援助热线
```
